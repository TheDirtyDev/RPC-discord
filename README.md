##  Features

- Real-time **player location** (street name) or **movement speed** in MPH  
- Displays **current server player count**  
- Supports **custom large and small images** with hover text  
- **Join Discord** button linking directly to your server invite  
- **Connect** button to join your FiveM server instantly  
- Easy to **configure for beginners**  

---

##  Requirements

- FiveM server  
- Discord Application with Rich Presence enabled: [Discord Developer Portal](https://discord.com/developers/applications)  
- Two unique images uploaded in Rich Presence (`large` and `small`)  

---
